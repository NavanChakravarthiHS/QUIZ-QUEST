# Teacher's Guide to QuizQuest

This comprehensive guide will help you make the most of QuizQuest as an educator. From creating quizzes to analyzing student performance, this document covers all the features available to teachers.

## Getting Started

### Account Registration
1. Navigate to the QuizQuest platform
2. Click on "Teacher Login"
3. Click "Sign up" if you don't have an account
4. Fill in your details and register as a teacher
5. Log in with your credentials

### Dashboard Overview
Upon logging in, you'll see your teacher dashboard which includes:
- List of quizzes you've created
- Option to create new quizzes
- Ability to edit or delete existing quizzes
- Access to student results and performance analytics

## Creating Quizzes

### Step 1: Access Quiz Creation
1. Log in to your teacher account
2. Click the "Create New Quiz" button on your dashboard

### Step 2: Quiz Details
Fill in the basic quiz information:
- **Title**: A clear, descriptive name for your quiz
- **Description**: Brief overview of what the quiz covers
- **Timing Mode**: Choose between:
  - **Total Time**: Set one timer for the entire quiz
  - **Per Question**: Set individual timers for each question

### Step 3: Timing Configuration
- For **Total Time Mode**: Set the total duration for the entire quiz (in minutes)
- For **Per Question Mode**: You'll set time limits for each question individually

### Step 4: Add Questions
Click "Add Question" to create questions. For each question, you'll need to specify:

1. **Question Text**: The actual question
2. **Question Type**:
   - **Single Choice**: Student selects one correct answer
   - **Multiple Choice**: Student selects all correct answers
3. **Points**: How many points this question is worth
4. **Time Limit** (Per Question Mode only): How much time students have for this question
5. **Options**: Add answer options and mark which ones are correct

### Step 5: Question Management
- **Reorder Questions**: Drag and drop to change question order
- **Edit Questions**: Click the edit icon to modify any question
- **Delete Questions**: Click the trash icon to remove questions
- **Preview**: Use the preview button to see how your quiz will appear to students

### Step 6: Publish Quiz
Once you're satisfied with your quiz:
1. Review all questions and settings
2. Click "Create Quiz" to publish it
3. Your quiz will now appear in your dashboard

## Managing Quizzes

### Viewing Your Quizzes
All quizzes you've created will appear on your dashboard with:
- Quiz title and description
- Number of questions
- Timing information
- Creation date

### Editing Quizzes
1. Find the quiz you want to edit in your dashboard
2. Click the "Edit" button (pencil icon)
3. Make your changes
4. Save the updated quiz

### Deleting Quizzes
1. Find the quiz you want to delete in your dashboard
2. Click the "Delete" button (trash icon)
3. Confirm deletion in the popup dialog
4. The quiz will be permanently removed

## Quiz Features Explained

### Flexible Timing Options
QuizQuest offers two timing modes to suit different assessment needs:

1. **Total Time Mode**:
   - One timer for the entire quiz
   - Students can navigate between questions freely
   - Ideal for comprehensive assessments

2. **Per Question Mode**:
   - Individual timers for each question
   - Students move sequentially through questions
   - Ideal for testing quick recall and decision-making

### Question Types
1. **Single Choice Questions**:
   - Students select exactly one correct answer
   - Award full points for correct answer, zero for incorrect

2. **Multiple Choice Questions**:
   - Students select all correct answers
   - Partial credit can be earned based on selection accuracy

### Point-Based Scoring
- Assign different point values to questions based on difficulty
- System automatically calculates total scores
- Detailed breakdown shows points earned per question

## Student Management

### How Students Access Quizzes
1. Students log in to their accounts
2. They can see all active quizzes in their dashboard
3. They click "Start Quiz" to begin
4. Quizzes can only be attempted once per student

### Monitoring Student Progress
- Students can only attempt each quiz once
- The system prevents duplicate attempts
- All attempts are logged with timestamps

## Security Features

### Anti-Cheating Measures
QuizQuest includes robust anti-cheating features:
- **Tab-Switch Detection**: Monitors when students switch to other tabs/windows
- **Warning System**: Students receive warnings after each tab switch
- **Auto-Submission**: Quiz is automatically submitted after 3 tab switches
- **Time Constraints**: Prevents unlimited time for answering questions

### Data Privacy
- Student responses are securely stored
- Only authorized teachers can view results
- Personal information is protected according to privacy policies

## Best Practices

### Quiz Design Tips
1. **Clear Instructions**: Provide clear, unambiguous questions
2. **Appropriate Difficulty**: Mix easy, medium, and hard questions
3. **Consistent Formatting**: Use consistent question formats
4. **Proofreading**: Check for typos and errors before publishing
5. **Logical Flow**: Arrange questions in a logical sequence

### Timing Recommendations
1. **Total Time Mode**: 
   - Allow 1-2 minutes per question on average
   - Include buffer time for navigation and review

2. **Per Question Mode**:
   - Simple recall questions: 30-60 seconds
   - Complex problem-solving: 2-5 minutes
   - Reading comprehension: 1-3 minutes

### Point Allocation
1. **Difficulty-Based**: More points for harder questions
2. **Importance-Based**: More points for key learning objectives
3. **Consistency**: Maintain consistent point values for similar question types

## Troubleshooting

### Common Issues and Solutions

1. **Students Can't See My Quiz**
   - Ensure the quiz is marked as "Active"
   - Check that the quiz was created successfully
   - Verify students are using the correct login credentials

2. **Quiz Timing Issues**
   - Double-check timing mode and duration settings
   - Consider time zones if students are in different locations
   - Test the quiz yourself to verify timing behavior

3. **Question Display Problems**
   - Check for special characters that might not render properly
   - Ensure all required fields are filled in
   - Preview the quiz to see how it appears to students

4. **Editing Restrictions**
   - Remember that students who have already attempted a quiz cannot see changes
   - Significant changes may require creating a new quiz version

### Getting Help
If you encounter issues not covered in this guide:
1. Check the browser console for error messages
2. Verify your internet connection
3. Try refreshing the page
4. Contact technical support if problems persist
