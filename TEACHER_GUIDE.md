# Teacher Guide

## Introduction

Welcome to QuizQuest! This guide will help you understand how to use the platform effectively to create engaging quizzes for your students.

## Getting Started

### Account Setup
1. Create a teacher account using the signup page
2. Verify your email address
3. Log in to access the teacher dashboard

### Dashboard Overview
- **Create Quiz**: Design and schedule new quizzes
- **My Quizzes**: View and manage your created quizzes
- **Question Bank**: Store and organize questions by subject
- **Analytics**: View performance reports and statistics

## Creating Quizzes

### Basic Information
1. **Title**: Give your quiz a clear, descriptive title
2. **Description**: Provide details about what the quiz covers
3. **Scheduled Date/Time**: Set when the quiz will be available
4. **Timing Mode**: Choose between total time or per-question timing

### Question Creation
1. **Manual Entry**: Add questions directly in the quiz creation form
2. **Question Bank**: Load questions randomly from your saved collection
3. **Question Types**: 
   - Single Choice (one correct answer)
   - Multiple Choice (multiple correct answers)
4. **Points**: Assign point values to questions
5. **Images**: Add visual elements to questions (optional)

### Using the Question Bank
1. Navigate to the Question Bank section
2. Add questions organized by subject
3. When creating or editing a quiz, toggle "Use Question Bank"
4. Select a subject and number of questions to load
5. Questions are randomly selected from your bank

### Question Bank Management

#### Adding Questions to the Bank
1. Go to the Question Bank section from your dashboard
2. Click the "Add Question" button
3. Fill in the question details:
   - **Subject**: Categorize the question by subject (e.g., Mathematics, Science)
   - **Question Text**: Enter the actual question
   - **Question Type**: Select Single or Multiple Choice
   - **Options**: Add answer options and mark the correct ones
   - **Points**: Assign point value for correct answer
   - **Image (Optional)**: Add visual elements if needed
4. Click "Add Question" to save

#### Editing Questions
1. Find the question in your Question Bank list
2. Click the "Edit" button next to the question
3. Make your changes
4. Click "Update Question" to save

#### Deleting Questions
1. Find the question in your Question Bank list
2. Click the "Delete" button next to the question
3. Confirm deletion when prompted

### Using Question Bank in Quiz Creation and Editing
1. When creating or editing a quiz, toggle the "Use Question Bank" switch
2. Select a subject from the dropdown (only subjects with existing questions will appear)
3. Specify how many questions you want to include (the system will randomly select this many questions)
4. Click "Load Questions" to populate your quiz with questions from the bank
5. Note: This will replace any manually added questions

## Best Practices

### Quiz Design Tips
1. **Clear Instructions**: Provide clear, unambiguous questions
2. **Consistent Formatting**: Use consistent question formats
3. **Proofreading**: Check for typos and errors before publishing
4. **Logical Flow**: Arrange questions in a logical sequence

### Timing Recommendations
1. **Total Time Mode**: 
   - Allow 1-2 minutes per question on average
   - Include buffer time for navigation and review

2. **Per Question Mode**:
   - Simple recall questions: 30-60 seconds
   - Complex problem-solving: 2-5 minutes
   - Reading comprehension: 1-3 minutes

### Point Allocation
1. **Importance-Based**: More points for key learning objectives
2. **Consistency**: Maintain consistent point values for similar question types
